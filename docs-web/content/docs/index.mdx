**Loclean** is a powerful Python library for local data cleaning and extraction using LLMs. It allows you to:

- Extract structured data from unstructured text with 100% schema compliance.
- Clean and normalize messy data in DataFrames.
- Scrub sensitive PII data locally without sending data to external APIs.
